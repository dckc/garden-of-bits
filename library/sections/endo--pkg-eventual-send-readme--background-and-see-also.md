> Abstract: Background pointers to the TC39 eventual-send proposal (this package is a forward-compatible shim) and to related-work links: handled-promise spec drafts, Causeway distributed debugger, CapTP family. The See Also section names sister packages: @endo/captp (transport), @endo/marshal (serialization), @endo/exo (class API).

## Background

This package implements the
[ECMAScript eventual-send proposal](https://github.com/tc39/proposal-eventual-send),
which provides native language support for eventual send operations.
