## Run the larger demo

Visit
[https://docs.agoric.com](https://docs.agoric.com/guides/getting-started/) for
getting started instructions.

TL;DR:

* `yarn link-cli ~/bin/agoric`
* `cd ~`
* `agoric init foo`
* `cd foo`
* `agoric install`
* `agoric start`

Then browse to http://localhost:8000
