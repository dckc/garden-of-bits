> Abstract: The endo repository's security policy. Two substantive sub-sections: supported-versions (security fixes only for the latest branch, since SES and Endo are under active development and security review) and coordinated-vulnerability-disclosure (Agoric HackerOne, encrypted email, Keybase; acknowledgement within one business day; up to 72 hours to validate; standard 90-day industry norm for vulnerability coordination, with intent to release faster).
