> Abstract: A small pointer-shaped README. ERTP is Agoric's digital-asset standard — a uniform way of transferring tokens and other digital assets in JavaScript. All assets follow the same security properties, transfer mechanics, and exposing patterns. The README is mostly external pointers to the ERTP guide (agoric.com/documentation/ertp/guide/) and the now-separate `@agoric/zoe` smart-contract framework package. AGENTS.md names ERTP as primary code.
