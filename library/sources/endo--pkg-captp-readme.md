> Abstract: The @endo/captp package README. CapTP is a JavaScript implementation of the Capability Transport Protocol — bidirectional capability-bearing message passing over an arbitrary transport with promise pipelining and consistent capability identity across vat boundaries. Three substantive H2s: Usage (the makeCapTP API), Loopback (in-process connection pattern), TrapCaps (synchronous trapping mechanism for debugging).
