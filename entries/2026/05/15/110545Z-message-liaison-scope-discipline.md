# Lesson: Honor specificity in file-scope instructions

When the user says "delete that one .vo file" they mean ONE file.
Do not generalize to "delete all .vo files" — that is a 6-minute
rebuild penalty.

If you don't know which file is the problem, ask. Do not widen the
scope unilaterally.

Applies to: every role that touches build artifacts.
